# Site AI Score

이 Replit workspace 자체가 프로젝트 루트입니다.

- 프로젝트 루트: `/home/runner/workspace`
- 별도의 중첩 프로젝트 폴더를 만들지 않습니다.
- 일반 소스 폴더(`client`, `server`, `shared`, `prisma`)는 프로젝트 루트 바로 아래에 둡니다.
- 실행: `npm run dev`
- 개발 포트: `5000`

상세 협업 규칙은 `AGENTS.md`, 전체 기능 기획은 `SITEAISCORE_FINAL_DEVELOPMENT_PROMPT.md`를 확인합니다.
